# Task: CODE FORMATTING (CAR-8)

## Linear Link
https://linear.app/cartai-shop/issue/CAR-8/formateo-de-codigo

## Description
This task aims to order and format the project code according to strict visibility and alphabetical rules, which will be applied once the current task (CAR-5/6/7) is finished.

## Formatting Rules
The class code (attributes and methods) must be ordered as follows:

1. **`public` attributes** alphabetically.
2. **`protected` attributes** alphabetically.
3. **`private` attributes** alphabetically.
4. **`public` methods** alphabetically.
5. **`protected` methods** alphabetically.
6. **`private` methods** alphabetically.

*Note: New rules will be added to this story as needed.*
